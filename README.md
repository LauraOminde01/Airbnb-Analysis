# Airbnb Market Analysis

A data analysis project exploring Airbnb listing data to understand pricing patterns, market trends, and customer behavior.

## Project Overview

This project analyzes three datasets:
- **Listings**: Property details, pricing, and host information
- **Reviews**: Customer review data over time
- **Calendar**: Availability and pricing trends

## Key Findings

- Market structure and room type distribution
- Pricing patterns across neighborhoods and property types
- Temporal trends in bookings and reviews
- Host performance metrics
- Correlation patterns between listing features

## Technologies Used

- Python 3
- pandas - Data manipulation and analysis
- matplotlib - Data visualization
- seaborn - Statistical visualizations
- numpy - Numerical computations

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Ensure CSV files are in the project directory:
   - `listings.csv`
   - `reviews.csv`
   - `calendar.csv`

3. Open and run `eda.ipynb` in Jupyter Notebook or JupyterLab

## Analysis Structure

1. Data Loading - Import and examine datasets
2. Data Cleaning - Handle missing values and data types
3. Exploratory Data Analysis - Visualizations and statistical analysis
4. Key Insights - Summary statistics and findings

## Results

The analysis reveals pricing dynamics, seasonal patterns, and performance metrics that can inform business decisions for hosts, property managers, and market analysts.
